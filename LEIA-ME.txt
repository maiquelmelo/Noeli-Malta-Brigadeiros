NOELI MALTA BRIGADEIROS E BEM CASADOS — VERSÃO 2.0

ESTRUTURA
- index.html: tela principal
- assets/logo.png: logo oficial do sistema
- assets/favicon.png: ícone do navegador
- assets/splash.png: imagem de abertura
- assets/produtos/: futuras fotos do cardápio
- css/style.css: visual do aplicativo
- js/app.js: funcionamento do aplicativo
- manifest.json: instalação no celular/tablet
- service-worker.js: funcionamento offline

COMO PUBLICAR NO GITHUB PAGES
1. Exclua ou substitua os arquivos antigos do repositório.
2. Envie TODO o conteúdo desta pasta, mantendo as pastas assets, css e js.
3. Aguarde o GitHub Pages atualizar.
4. No celular, remova a versão antiga instalada.
5. Limpe os dados/cache do site no Chrome.
6. Abra novamente o endereço e instale o aplicativo.

COMO TROCAR A LOGO NO FUTURO
Substitua apenas o arquivo assets/logo.png, mantendo exatamente o mesmo nome.
Para o ícone mudar imediatamente, substitua também assets/favicon.png e assets/splash.png e altere a versão do CACHE no service-worker.js.

DADOS
Os pedidos ficam salvos no próprio aparelho. Faça backups regularmente pelo menu Mais.
